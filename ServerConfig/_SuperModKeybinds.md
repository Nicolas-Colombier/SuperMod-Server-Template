# ⌨️ KEYBINDS

## 🔫 Tilt weapon :
* Press **Switch to item 7** : Tilt right
* Press **Switch to item 7** twice : Tilt left

## 🔫 Buildable foxhole :
* Press **Middle Mouse Button** when selecting the shovel : add a foxhole to your inventory (then place it and build it)

————————————————————————————

## 🔩 Auto Zeroing for MBT (need stabilization ON) :
* Press **C** : Set the range
* Press **Left Ctrl + C** : Reset the range

## 🔩 Manual Zeroing for Leclerc / T-62 / T-64BV / BMD-2M / BMP-2 / T-72A / T-55 :
* Press **Start ranging + mouse scroll** : Set the range

## 🔩 Manual Zoom for T-64BV :
* Press **Left Ctrl + mouse scroll** : Set the zoom

## 🔩 Switch to Periscope view for MBT :
* Press **Left Shift** : Activate/Deactivate
* Press **Left Alt** : Freelook

## 🔩 Switch between Periscope views for Leclerc :
* Press **Left Ctrl** : Cycle

## 🔩 Change sight color (red/black) for Leclerc / T72A :
* Press **P** : Cycle

## 🔩 Adjust suspension for AMX 10 RCR :
* Press **Space bar +  mouse scroll** : Adjust suspension pitch
* Use **Mouse scroll** : Adjust suspension elevation and roll

————————————————————————————

## 🚁 Switch to cockpit view for CAS gunner :
* Press **Left Shift**
* Press **Left Alt** : Freelook

## 🚁 HMD Auto Zeroing for EC665 gunner in cockpit view :
* Press **X** : Set the range
* Press **Left Ctrl + X** : Reset the range

## 🚁 Use sight zoom for CAS pilot :
* Press **Aim Down Sights** : Cycle

## 🚁 Use flares for helicopters :
* Press **Switch to item 3**: Release a single flare
* Long press **Switch to item 4** : Release multiple flares
